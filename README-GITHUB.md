# São José Peregrinações — cópia para GitHub Pages

Esta pasta foi adaptada para hospedagem estática no GitHub Pages.

- Publique os arquivos na raiz do repositório.
- Em Settings > Pages, selecione a branch e a pasta /(root).
- O arquivo `.nojekyll` deve permanecer na raiz.
- Esta versão usa links `.html`, pois o GitHub Pages não executa `.htaccess`.
- O site oficial na HostGator não deve ser substituído por esta cópia.
