# São José – Artigos Religiosos e Peregrinações

Website institucional desenvolvido para a **Loja São José**, empresa com mais de 20 anos organizando peregrinações católicas, sediada em Campo Grande, Mato Grosso do Sul, Brasil.

## Sobre o Projeto

Site completo com 8 páginas, totalmente responsivo, focado em geração de contato via WhatsApp.

**Páginas:**
- Início
- Sobre Nós
- Peregrinações (6 destinos)
- Detalhe de Peregrinação (template com roteiro de 10 dias)
- Calendário 2026
- Parceria Canção Nova
- Galeria
- Contato

**Tecnologias:**
- HTML5 / CSS3 / JavaScript (vanilla)
- Google Fonts (Cinzel, Playfair Display, Lato)
- Totalmente responsivo (mobile, tablet, desktop)
- Sem frameworks, sem dependências externas
